Introduction to Java

Introduction

Java, is a powerful yet simple High-Level programming language. It is a language that allows its users to build powerful codes that could be used to perform tasks varying from simple calculations to creating complex websites and applications. It is a great start for beginners who are looking forward to diving into the field of computer sciences and are searching for their first step towards it. Besides its simplicity, Java offers numerous tools that help developers save their time and energy by offering various built-in features which we'll discuss later in this blog.

History and Development

Java was developed by the Sun Microsystems headed by James Gosling in the year 1991 and it was initially named 'Oak'. The name 'Oak' was however discarded as it was taken by some other language that arrived before this language and so finally later on the name 'Java' came into existence. The languages used in the world before Java such as C or C++ were sufficient but however there was a lot of load on the programmer due to the complexity of these languages. This complexity not only reduced the productivity of these languages but also required more time. Moreover, error detection and error fixing was very difficult making the programmers look forward to a language that could solve these problems. Initially, Java was designed just to create internet applets that run on web browsers but due to its advanced features and simplicity it soon became popular and was then used as a programming language that was accepted worldwide.

Features of Java

Java offers various features that help its users to minimize their workload, save time, and give efficient results desired by the programmers. A few of the features offered by Java are:

  • Java is an Object Oriented Programming Language (OOP).
  • Java is a platform-independent programming language i.e. unlike earlier languages that operated only on the system they were written on, it could run on any system that had Java installed.
  • It is a WORA (Write Once Run Anywhere) programming language.
  • It is a multithreaded language.
  • Java is case sensitive i.e. Uppercase and Lowercase letters are treated differently.
  • Being an OOP language, the code is written within a class and all the variables, methods/functions, comments, and all the related elements are contained in this class.
  • Java can be used to create both Internet Applets that run on Web Browsers as well as Applications that can run on computers just like other programming languages.

Comments in Java

The comments are lines or pieces of text that are written in a program but are not executed by the Java compiler. The purpose behind writing comments is for the sake of understanding the code later on by the programmer or any other user. These comments can be written in two ways:

  1. Single line comments: Only a single line can be written after using //.
    For example: //This is a sample single line comment
  2. Multi line comments: Here multiple lines or paragraphs can be written between /* ... */.
    For example: /*This is a sample
    multi line comment*/

Tokens

The smallest or the fundamental unit of a program is called a token. It is every individual constituent that constitutes a program of Java. The tokens in Java are:

  • Literals (Constants)
  • Identifiers
  • Punctuators
  • Operators
  • Separators
